# 임상 사진 워터마크 프로그램

## 1. 프로그램 목적
임상 사진에 병원 로고와 촬영일자, 강조 요소(박스/화살표)를 빠르게 넣고 여러 장을 한 번에 일괄 저장할 수 있는 툴

---

## 2. 기본 사용 프로세스

### 1) 치료 사례 이미지 여러 장 불러오기
- JPG, PNG 등 일반 이미지 포맷 지원
- 해상도/사이즈가 서로 다른 이미지도 문제 없이 적용

### 2) 로고 & 촬영일자 입력
- 병원 로고 이미지 1개 업로드 (PNG 권장, 투명 배경 지원)
- 촬영일자 텍스트 입력 (예: 22.03 형식)
- 필요 시 글꼴, 글자 크기, 색상 기본값 설정 가능하면 좋겠습니다.

### 3) 배치 위치 선택
- 로고 위치: 이미지 위에서 드래그해서 위치 지정 (좌/우 상단, 하단 등 자유 배치)
- 촬영일자 위치: 로고와 함께 또는 별도의 위치로 지정 가능하도록
- 한 번 위치를 정하면, 같은 세션에서 불러온 모든 이미지에 동일한 위치로 적용되면 좋겠습니다.

### 4) 강조 요소 설정 (선택 기능)
- 이미지 위 특정 부위를 점선 박스/실선 박스/화살표 등으로 표시
- 박스 색상 및 두께, 사이즈, 모서리 라운드 여부 선택 가능
- 이미지에 간단한 짧은 텍스트를 넣을 수 있으면 좋겠습니다.
- 강조 박스도 위치를 자유롭게 움직여서 배치할 수 있도록

### 5) 저장
- 로고 + 촬영일자 + 강조 요소가 적용된 최종 이미지를 한 번에 일괄 저장 하는 기능
- 원본은 그대로 두고, 저장 폴더/파일명 설정 가능 (예: 섬유종_1.jpg)

---

## 3. 추가 희망 사항

- 다양한 사이즈/비율의 이미지에서도 깨지지 않고 자연스럽게 적용될 것
- 최종 이미지 품질 저하 최소화
- 추후 로고 교체(색 변경), 글꼴 변경, 기본 위치값 수정이 가능하면 좋겠습니다.
